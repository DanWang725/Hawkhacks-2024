## Getting Started

### Installing Python Libraries

Run the commands

```bash
cd backend
python -m venv venv
./venv\Scripts\activate
pip install -r requirements.txt
```

## Running Dev Servers

### Frontend

```bash
yarn start
```

### Backend

```bash
cd backend
uvicorn main:app --reload
```
